# Databricks - Aula 03

## Descrição
Como criar e gerenciar objetos no Databricks, incluindo Catálogos, Schemas e Tabelas.

## Conteúdo
- Criação de Catálogo
- Criação de Schema
- Criação de Tabela
- Manipulação de dados
- Exemplos de comandos SQL e Python

## Requisitos
- Acesso ao Databricks
- Permissões para criar objetos no workspace
